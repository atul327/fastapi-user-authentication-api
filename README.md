# FastAPI User Authentication System

This project is a **Full Stack User Authentication System** built using **FastAPI (Backend)** and **Streamlit (Frontend)**.

## 🧰 Tech Stack

* **FastAPI** – Backend API
* **Streamlit** – Frontend UI
* **MySQL** – Database
* **Railway** – Backend Hosting
* **Streamlit Cloud** – Frontend Hosting
* **GitHub** – Version Control

---

## ✨ Features

* User Registration
* User Login
* Password Reset
* File Upload
* Secure Password Hashing
* REST API with FastAPI
* Interactive UI with Streamlit

## ⚙️ Installation (Local Setup)

Clone the repository:

```
git clone https://github.com/atul327/fastapi-user-authentication-api.git
```

Install dependencies:

```
pip install -r requirements.txt
```

Run FastAPI:

```
uvicorn main:app --reload
```

Run Streamlit:

```
streamlit run frontend.py
```
